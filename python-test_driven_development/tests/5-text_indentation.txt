========================================
How to Use 5-text_indentation
========================================

Import the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

1. Normal Usage:
================
    >>> text_indentation("Holberton. School? is: awesome.")
    Holberton.
    <BLANKLINE>
    School?
    <BLANKLINE>
    is:
    <BLANKLINE>
    awesome.
    <BLANKLINE>

    >>> text_indentation("   Leading and trailing spaces.   ")
    Leading and trailing spaces.
    <BLANKLINE>

2. Exceptions (Errors):
=======================
Testing when text is not a string:
    >>> text_indentation(123)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

    >>> text_indentation(["List", "of", "strings"])
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

    >>> text_indentation(None)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

3. Edge Cases:
==============
Missing arguments:
    >>> text_indentation()
    Traceback (most recent call last):
        ...
    TypeError: text_indentation() missing 1 required positional argument: 'text'

Empty string:
    >>> text_indentation("")

No punctuation:
    >>> text_indentation("Hello world")
    Hello world

Multiple delimiters:
    >>> text_indentation("Hello..")
    Hello.
    <BLANKLINE>
    .
    <BLANKLINE>
