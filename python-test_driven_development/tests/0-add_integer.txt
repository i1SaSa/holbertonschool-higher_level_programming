======================================
How to Use 0-add_integer
======================================

This module defines a function ``add_integer(a, b=98)``.

First, we need to import the function:
    >>> add_integer = __import__('0-add_integer').add_integer

1. Normal Integers Addition:
============================
    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98
    >>> add_integer(0, 0)
    0
    >>> add_integer(-5, -5)
    -10

2. Default Value for b:
=======================
If we only pass one argument, b defaults to 98:
    >>> add_integer(2)
    100
    >>> add_integer(-2)
    96

3. Float Addition (Casting to int):
===================================
Floats should be cast to integers before addition:
    >>> add_integer(100.3, -2)
    98
    >>> add_integer(100.9, -2)
    98
    >>> add_integer(10.5, 10.5)
    20
